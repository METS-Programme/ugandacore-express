Introduction

 Welcome to OpenMRS Express:HIV. This version of OpenMRS is the complete standard 1.1 OpenMRS with the addition of specific content to implement the World Health Organization's patient monitoring guidelines for HIV care and antitretroviral therapy. This package is currently beta software. If you would like to participate in the testing of this implementation in a clincal setting, please contact us (see Contact and support) More detailed information on OpenMRS Express:HIV can be found on the OpenMRS website, http://www.openmrs.org and on the OpenMRS Express:HIV page.
There is also a support group, openmrs-express, available here: http://groups.google.com/group/openmrs-express.

Installing

Double click on "Install.exe" to un archive the installer. Double click then on the Install.bat script to run the installer. It will install the various components used by OpenMRS Express:HIV, as well as control scripts which will be adapted to your machine in order to allow you to start and stop the application as well as connect to it with your web-browser. Should windows display security warnings during the installation, please allow the requested operations.

Running and Stopping OpenMRS Express:HIV

The OpenMRS server is started automatically at your computer startup.

You can manually start or stop OpenMRS server by double clicking on

  • $OPENMRS_HOME\script\start_openmrs.bat
  • $OPENMRS_HOME\script\stop_openmrs.bat

If you chose "Shortcut in Windows Start menu", these script can be accessed from Start Window Menu > Programs > OpenMRS > Admin

$OPENMRS_HOME should be C:\Program Files\WHO\OpenMRS

Connecting to OpenMRS Express:HIV

 If you chose "Shortcut in Windows Start menu" option during the installation process, click on the file Start Window Menu > Programs > OpenMRS > Access OpenMRS

Once you have OpenMRS Express up and running, you can always reconnect to it by accessing the following link with your web browser:

http://127.0.0.1:{tomcat.port}/openmrs

{tomcat.port} default value is 8079, so the URL should be http://127.0.0.1:8079/openmrs

First time configuration

 The first time you run OpenMRS Express:HIV, you should perform some simple configuration steps:
  1. Set the name of your facility:
    1. Log in as admin, the default password is test.
    2. Click on Administration at the top right of the window.
    3. Under Encounters, click on Manage Locations.
    4. Click on My Facility.
    5. Fill in the form on the screen. Make sure to replace the string "My Facility" in the Name field to the name of you facility.


  2. Create user accounts for care providers.
    1. Click on Administration on the top right of the window.
    2. Under Users, click on Manage Users
    3. Click on the Add User link.
    4. Fill in the form, then click on the Create Person button.
    5. Fill in the detailed form, and select OpenMRS Express User for the role, then click on the Save User button.

  3. Log in with your new user account:
    1. Click on the Log out link in the top right of the window.
    2. Enter your new username and password in the login form, then click the Log In button.

Infopath 2003 support

 From release 5, OpenMRS Express:HIV includes Infopath 2003 forms. These forms are generated automatically from the Infopath 2007 versions and are constrained by Infopath 2003 functional limitations, particularly in the area of date manipulation. These forms are not fully supported or tested so the default installation disables them. If you would like to enable or use these forms, check the "Enable infopath 2003" checkbox during the installation process.

Using OpenMRS Express

 In order to enter data and view individual patient information, log in using a provider account like the one you created during the first time configuration above.

In order to use any of the administration functions (under the Administration heading) you should log out of a provider account and login using the administrative account, admin (see password below)

Troubleshooting

Installation Issues

  • Windows may display a Security Warning when running the install. If it does so, please UNCHECK the "Always ask before opening this file" option, then, click on Run. This will allow the installation script to complete its task.
  • The install script will activate the database server in order to load the initial OpenMRS Express:HIV database and parameters. If you are using a firewall, it is possible that you will be asked whether or not to allow a program to connect or listen. If this is the case, allow the connection to go through.

Running OpenMRS Express:HIV

  • If you are using a firewall, when starting OpenMRS Express:HIV, you may be notified that a process is trying to connect or listen to the network. If this occurs, allow the connection to go through.
  • If you get an error when attempting to connect to OpenMRS Express with your browser for the first time, wait a few moments for the server to startup then refresh the page.

Software components

 This distribution package contains applications and programs from a variety of sources. The packages includes the following:
  • Java 6, update 5
  • Apache Tomcat 6.0.16
  • Eclipse BIRT 2.2.2
  • MySQL 5.0.51a
  • Super-Sed 3.62
  • Microsoft sleep utility
  • OpenMRS core 1.3
  • birt-1.8 module
  • clinicalsummary-3.0 module
  • formentry-3.2 module
  • WHOmod 0.15 module
If you notice a component included in this distribution which is not present in this list, please let us know so that we can update it.
